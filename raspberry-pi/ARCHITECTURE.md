## CSS Touch Optimizations

```css
/* Global touch optimizations */
* {
  -webkit-touch-callout: none;      /* No iOS callout menu */
  -webkit-user-select: none;        /* No text selection */
  user-select: none;
  -webkit-tap-highlight-color: transparent; /* No tap highlight */
}

/* Allow selection in inputs */
input, textarea {
  -webkit-user-select: text;
  user-select: text;
  font-size: 16px;                  /* Prevents zoom on focus */
}

/* Touch-friendly sizes */
button {
  min-height: 48px;                 /* Apple/Google guidelines */
  min-width: 48px;
  touch-action: manipulation;       /* Disable double-tap zoom */
}

/* Smooth scrolling */
body {
  -webkit-overflow-scrolling: touch;
  overscroll-behavior: contain;     /* Prevent pull-to-refresh */
}
```
